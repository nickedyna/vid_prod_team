# Asset Naming Guide

The video JSON references b-roll files by path. Keep asset names stable so the automation does not break.

## B-roll clips

Store reusable clips in:

```text
assets/broll/
```

Use lowercase snake case names:

```text
job_board_scroll.mp4
campaign_brief_scroll.mp4
html_email_code.mp4
email_preview_desktop_mobile.mp4
chatgpt_agent_prompt.mp4
template_generator_output.mp4
qa_checklist_scroll.mp4
zip_package_folder.mp4
corporate_desk_setup.mp4
avatar_ai_email_dev_loop.mp4
subscribe_end_card.mp4
```

## Voiceover

Store voiceover files in:

```text
assets/voiceover/
```

Recommended format:

```text
brief-to-zip_01_voiceover.wav
brief-to-zip_01_voiceover.mp3
```

## Avatar

Store the main avatar image in:

```text
assets/avatar/avatar.png
```

Transparent PNG is preferred.

## Music

Store royalty-free music in:

```text
assets/music/
```

Recommended format:

```text
corporate_tech_pulse_01.mp3
```

Only use music you have the rights to use commercially.
